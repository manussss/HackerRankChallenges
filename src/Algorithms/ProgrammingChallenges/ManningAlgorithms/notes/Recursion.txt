-> Caso base e caso recursivo

-> Caso recursivo é quando a função chama a si mesma.
-> Caso base é quando a função não chama a si mesma novamente, de forma 
que o programa não se torne um loop infinito.

-> Tail Recursion: Tail recursion is just a particular instance of recursion, 
where the return value of a function is calculated as a call to itself, and nothing else.
